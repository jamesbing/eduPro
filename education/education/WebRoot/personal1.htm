Cache Scheme

 

 

Direct Mapped Cache:Block is placed according to the formula

Frame Number mod Cache size

 

Set Associative Cache:Block is placed according to the formula

Block number mod Number of Sets

 

 

For each Cache Scheme, a Table depicting the structure of the Cache will be shown.

In the Main Memory Frame (to the Right), the user can enter binary or decimal address and the corresponding cache location will be displayed along-with a break up of the address structure.
